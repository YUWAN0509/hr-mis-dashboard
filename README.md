# HR MIS Dashboard

## 📌 Project Overview
This project is an HR MIS Dashboard developed using Microsoft Excel to manage and analyze employee information, attendance, payroll, and HR reports.

## 🚀 Features
- Employee Master Data Management
- Daily Attendance Tracking
- Monthly Attendance Summary
- Payroll Calculation
- Salary Slip Generation
- Pivot Tables & Charts
- Interactive HR Dashboard

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Formulas
- Conditional Formatting
- Data Validation
